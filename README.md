# 🛡️ Seguridad Informática – IFCT0109

¡Hola! 👋 Este repositorio representa mi formación completada en el **Certificado de Profesionalidad de Seguridad Informática (IFCT0109)**, conforme al RD 686/2011 y su modificación RD 628/2013.

A lo largo de este programa he adquirido las competencias necesarias para garantizar la seguridad de accesos, uso de la información, protección de sistemas y comunicaciones, tanto a nivel interno como externo de una organización.

---

## 🧠 Competencias adquiridas

Este certificado se basa en la cualificación profesional **IFC153_3 Seguridad Informática**, compuesta por las siguientes unidades de competencia:

- 🔐 **UC0486_3**: Asegurar equipos informáticos  
- 🕵️ **UC0487_3**: Auditar redes de comunicación y sistemas informáticos  
- ⚠️ **UC0488_3**: Detectar y responder ante incidentes de seguridad  
- 🔑 **UC0489_3**: Diseñar e implementar sistemas seguros de acceso y transmisión de datos  
- 🧰 **UC0490_3**: Gestionar servicios en el sistema informático

---
