Últimos cambios

Historial de los cambios

NoteBookCast esta continuamente mejorando y añadimos nuevas características acorde de las sugerencias de nuestros visitantes.
8.5.0 - December 28 ,2024
  • Now you can rotate embedded text and choose a font style.
  • Fix the pdf whiteboard generator to include the tables and image type backgrounds.
8.4.0 - November 7 ,2024
  • Added a button to delete embedded texts, keeping that original behaviour of deleting a text when its empty.
  • Improve the interface Videos windows of a video call, giving the freedom to drag the video window any where in the whiteboard, resize and shape as preferred, the new interface allows also to fit many more visible video windows.
8.3.0 - September 15 ,2024
  • Change the behaviour of shapes drawing, no need to confirm each shape.
  • Improve screen sharing stability.
8.2.0 - July 25, 2024
  • Synchronized timer added.
  • Draw tables.
8.1.0 - December 10 , 2023
  • Full screen option added (whiteboard bottom menu).
  • Fast image upload option (whiteboard left menu).
  • Use webcam to add picture to whiteboard.
  • Scanner mode for pictures.
8.0.0 - September 1 , 2023
  • Share screen enabled with high quality view.
  • Annotation and drawing above the shared screen.
  • Fast eraser in the shared screen view.
  • Night mode synchronization.
7.0.0 - June 25 , 2023
  • Unlimited audio call, supports all users in the whiteboard.
  • Support speedy image paste in the whiteboard.
  • Improve the new interface usability.
6.1.0 - November 7, 2022
  • Improve speed of realtime cursors broadcast.
  • Night mode.
  • Move the close whiteboard button to the bottom menu.
6.0.0 - July 19, 2022
  • Realtime cursors broadcast.
  • Follow me feature.
  • Everyone follows feature.
  • Improved ruler angle indicator as suggested by users.
  • Improved load speed of whiteboards.
  • Improved whiteboard capture and social share page generation.
  • Added new servers in the US.
5.4.0 - March 21, 2022
  • Social share integration and implementation.
5.3.0 - March 10, 2022
  • Added the ability to edit integrated texts.
5.2.0 - February 14, 2022
  • New Ruler tool.
  • Allow resize whiteboard after creation.
  • Added Favorites functionality for menu elements.
  • Refactor and improve the move images functionality.
  • Synchronise move images between participates in real time.
  • Multiple new whiteboard settings options.
5.1.0 - December 1, 2021
  • New settings for page synchronisation between participants. More info
  • Fix a bug reported related to image move.
5.0.0 - November 6, 2021
  • Integration of Audio call.
  • Added audio notes support (in assets manager).
4.1.0 - October 4, 2021
  • Enable image moving once posted to the whiteboard.
  • Optimize and speed up images loading in whiteboard.
4.0.0 - August 22, 2021
  • Multi-page whiteboards support.
  • Repeatable shapes feature.
  • Now the paper background is synchronized in real time.
  • New bottom menu for pages management and chat.
  • Whiteboards QR code for easy sharing and joining the whiteboards.
  • Added ten new servers in NewYork region.
  • Date and time for chat messages and assets list now shows according to visitors local time.
  • Add the ability to copy and paste inside a sticky note.
  • Minor performance improvements and fixes.
3.3.0 - April 11, 2021
  • Added two new shapes, arrows and double arrows.
  • Sticky notes improvements.
  • Now its possible to clear only specific images from the whiteboards.
3.2.0 - March 8, 2021
  • Improved draw smoothing engine for better draw and write quality.
  • Added an extra thin pen width.
  • Added sticky notes feature.
  • Some minor fixes for better connectivity stability.
  • New servers added.
3.1.0 - November 15, 2020
  • New whiteboard capture engine.
  • Fix a bug related to the laser pointer not having user assigned colour.
  • Some improvements to increase whiteboard performance.
3.0.0 - November 02, 2020
  • Brand new design with improved usability and mobil support.
  • Whiteboard improved mobil and tablet support.
  • Whiteboard one click text add, improved usability for text adding.
  • A new button added to the whiteboard for mobil devices allowing to drag the whiteboard easily.
  • The paper background of the whiteboard captures has been removed.
